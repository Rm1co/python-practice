### 1. factorial_loop.py
- **Description**: Calculates the factorial of a number using an iterative approach (for loop).
- **Function**: `factorial_loop(n)`
- **Test Cases**:
  - Factorial of 5 (output: 120)
  - Factorial of 4 (output: 24)

### 2. factorial_recursive.py
- **Description**: Calculates the factorial of a number using recursion.
- **Function**: `factorial_recursive(n)`
- **Test Cases**:
  - Factorial of 6 (output: 720)
  - Factorial of 2 (output: 2)

### 3. even_or_odd.py
- **Description**: Determines whether a number is even or odd using modulus operation.
- **Function**: `check_even_or_odd(num)`
- **Test Cases**:
  - Number 22 (output: "Even")
  - Number 19 (output: "Odd")

### 4. reverse_string.py
- **Description**: Reverses a given string using string concatenation in a loop.
- **Function**: `reverse_string(s)`
- **Test Cases**:
  - String "hello" (output: "olleh")
  - String "Michael" (output: "leahciM")

### 5. sum_list.py
- **Description**: Calculates the sum of all elements in a list of numbers.
- **Function**: `sum_list(numbers)`
- **Test Cases**:
  - List [7, 8, 9, 10] (output: 34)
  - List [50, 60, 70] (output: 180)
  - List [-5, 15, 25] (output: 35)

### 6. sum_of_digits.py
- **Description**: Calculates the sum of all digits in a given integer.
- **Function**: `sum_of_digits(n)`
- **Test Cases**:
  - Number 789 (output: 24)
  - Number 512 (output: 8)
